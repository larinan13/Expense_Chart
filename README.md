# Expense Chart - Личный финансовый трекер

**Автор:** Анастасия Ларина

---

## Описание программы

Expense Chart — это консольное приложение для учёта личных расходов. Позволяет добавлять записи о тратах по категориям (Еда, Транспорт, Развлечения и др.), фильтровать их по категории или периоду, подсчитывать сумму расходов за выбранный интервал и строить наглядные графики с помощью библиотеки matplotlib. Подходит для ежедневного контроля бюджета и анализа структуры расходов.

---

## Требования для запуска

- Python 3.8 или выше
- Библиотека: `pip install matplotlib`

---

## Как запустить

```bash
git clone https://github.com/larinan13/Expense_Chart.git
cd expense-chart
pip install -r requirements.txt
python src/main.py
```
